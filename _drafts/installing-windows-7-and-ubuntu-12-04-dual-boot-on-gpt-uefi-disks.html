This is probably very important for all those who have newer laptops and are FORCED to use UEFI - plus have larger disks (>2tb) that wont obey MBR boot records.

Preliminaries

if you have multiple hard disks, please remove the one that is not the install target. Windows screws up royally here. This wasted more than a day of my time.

First install linux

This is how my partition table looks like. Note that the special "bios_grub" flag is important to recover linux grub after installing windows. This flag can only be set from commandline using sudo parted /dev/sda set 1 bios_grub on

IMPORTANT: Create a 100 mb fat32 partition as the first partition. I'm not sure what is going on, but without that partition my GPT setup doesnt work.
